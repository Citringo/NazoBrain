NazoBrain
========

NazoBrain は，日本語での会話のようなものができるかもしれないAIのようなものです．

現状では，お勉強タブにテキストを流し込むことによって，会話を学習します．

改行で区切った文字列の前後を関連付けることにより，会話への返答方法を学習します．

文字列の形態素解析にMeCabの.NET移植版「NMeCab」を使用しています．

Build
----------
Visual Studio 2017 でソリューションを開いてビルドしてください．対応するMSBuildでもビルド可能です．
Nuget を使用しているのでパッケージを復元する必要があります．
